# Ruhan Portfolio v3 — Upgraded

## What's new
- **Vertical sidebar navigation** with dot indicators + tooltip labels + social icons
- **Horizontal section transitions** — sections slide left/right (Framer Motion `AnimatePresence`)
- **Custom cursor** with smooth follower and hover expansion
- **Typewriter effect** in hero subtitle
- **Animated skill bars** that fill on scroll into view
- **Animated counters** (stats on hero)
- **Progress bar** at top showing which section you're on
- **Keyboard navigation** — arrow keys move between sections
- **Scroll wheel** navigation on non-scrollable sections
- **Section label** bottom-right shows current section
- **Syne + DM Sans** fonts for a distinctive editorial feel
- **Noise overlay** for texture depth
- **Ambient orbs** in hero and contact sections
- **Enhanced project cards** with color accents per project
- **Experience cards** with tech tags
- **Mobile nav bar** at bottom for small screens

## Install & Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
